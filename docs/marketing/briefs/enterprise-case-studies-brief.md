## Brief Template — Per Case Study

For each candidate customer, Research Lead provides:

### 1. Customer Name
- Must be publicly referenceable (signed reference permission or public quote)
- If not yet cleared: use "Fortune 500 financial services firm" style anonymization until cleared
- Priority: named > quote > anonymized

### 2. Use Case
- What problem were they solving?
- What Molecule AI features did they deploy?
- What team size / agent count?

### 3. Deployment Model
- Self-hosted, Molecule AI Cloud, or hybrid?
- Which backend (Docker, Fly.io, Control Plane)?
- Integration stack (which MCP servers, which APIs)?

### 4. Measurable Outcome
- Time saved, cost reduction, team scale, incident reduction
- Any quantified metrics the customer approved for publication
- Even directional ("reduced agent setup time by ~70%") is useful if precise metrics aren't cleared

### 5. Pull Quote (if available)
- One sentence from the customer, approved for attribution
- If not yet cleared: can anonymize to role ("Senior Platform Engineer, enterprise fintech")
