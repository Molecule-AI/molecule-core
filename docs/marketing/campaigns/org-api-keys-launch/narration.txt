The days of one shared admin key — full access, no audit trail — are over.

Molecule AI org-scoped API keys let platform admins mint named tokens,
scoped read or write, with expiry dates and instant revocation.

Open the Molecule AI Canvas, go to Org Settings, and click Create new token.
Give it a name — maybe "ci-pipeline-key" — set the scope to write,
set expiry to ninety days, and create it.

The plaintext token is shown exactly once. Copy it now, or it's gone.

From your CI pipeline, authenticate with that token. Attach a Cloudflare
Artifacts repo, query workspace state, check activity logs — all scoped
to your org, nothing outside it.

Someone leaves the team. You go back to Org Settings, find the token,
and click Revoke. The token dies immediately. No grace period, no cached
sessions. It is gone.

Org-scoped API keys: mint from the canvas, use in your integrations,
revoke in one click. One key per integration — and you always know
exactly who's using what.
